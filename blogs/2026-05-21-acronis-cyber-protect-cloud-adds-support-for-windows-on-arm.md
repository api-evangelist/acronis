Windows on ARM is becoming increasingly relevant for business endpoints. Newer ARM-based Windows laptops are built for mobility, long battery life, quiet operation and on-device AI workloads. Acronis Cyber Protect Cloud now supports Windows on ARM devices for the core services MSPs need first: Backup, anti-malware, self-protection, remote management and cyber scripting.
